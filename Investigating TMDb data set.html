INTRODUCTION¶

DATASET DESCRIPTION¶

The data set I opted to analyse is the TMDb movies data set. The data set contains a list of movies with data such as popularity, it's release year, popularity,revenue, voting, budget etc which will help in the analysis.

The data set has different variables for analysis and for the purposes of this project I focused on analysing the following trends;

  1. Movie Runtime over the given period
  2. Movie release trend over the given period
  3. Movie popularity over the given period
  4. Generated movie revenue over the given period

I also sought to analyse how different variables affect the generation of revenue as this is the main goal and crux of every business especially the movie making business. I chose the below listed variables to analyse the correlation;

  1. Revenue and Popularity
  2. Revenue and Runtime
  3. Revenue and Budget
  4. Revenue and Vote Count

The analysis was performed by using Python, in the below code cell,I wrote statements to import the packages - numpy and pandas which has functions that were used in the analysis. I also included the statements for visualization of the analysis and communicated the findings using plots.

For Exploratory Data Analysis, I split my analysis into two research questions as shown below.

RESEARCH QUESTION 1¶

Over the given period,what was the trend of the number of movies released, popularity of the movies released, runtime of the movies released and revenue generated?

In order to answer the above questions, I created a new data frame to have the data grouped by release year with functions size since I intended on finding out the number of movies released over the years and to also find and review the trends on revenue, runtime and popularity, I grouped the data by release year with the function mean to obtain the summary statistics on mean of the values over the years.

RESEARCH QUESTION 2¶

How is Revenue Correlated with the variables - Popularity, Vote count, Budget and Runtime?

def movies_function(flag,xlabel, ylabel, title):
    Movie_trend = TMDb_data.groupby('release_year').size()
    Mean_trend = TMDb_data.groupby('release_year').mean()
    if flag == 1:
        Release_Trend = Movie_trend.plot.line(color='purple')
    elif flag == 2:
        Popularity_Trend = Mean_trend['popularity'].plot.line(color='purple')
    elif flag == 3:
        Runtime_Trend = Mean_trend['runtime'].plot.line(color='purple')
    elif flag == 4:
        Revenue_Trend = Mean_trend['revenue'].plot.line(color='purple')
    plt.xlabel(xlabel)
    plt.ylabel(ylabel)
    plt.title(title)
